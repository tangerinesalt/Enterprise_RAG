## 知识库模块

### 列表页（`/kb`）

```
┌──────────────────────────────────────────────────┐
│  知识库                               [+ 新建]   │
├──────────────────────────────────────────────────┤
│                                                  │
│  📁 my-docs                        3 文件 1 文件夹│
│  📁 test-kb                        2 文件        │
│  📁 rag-docs                       0 文件        │
│                                                  │
│         [点击进入详情 →]                          │
│                                                  │
└──────────────────────────────────────────────────┘
```

- 点击知识库行 → 跳转 `/kb/{name}`
- 每行右侧删除按钮（🗑️）
- 新建弹窗输入名称

### 详情页（`/kb/{name}`）

```
┌──────────────────────────────────────────────────┐
│  ← 返回                                   知识库名│
├──────────────────────────────────────────────────┤
│  📄 sample.txt                    2.5KB  [🗑️]   │
│  📄 report.pdf                  325KB   [🗑️]   │
│  📁 my-folder/           2 files         [🗑️]   │
│                                                  │
│  [上传文件]  [上传文件夹]  [索引全部]              │
│  ── 点击文件/文件夹行末的 [索引] 可单独索引 ──    │
└──────────────────────────────────────────────────┘
```

- 每行右侧：删除按钮，索引按钮（文件夹索引其下所有）
- 上传按钮触发文件选择对话框

## 会话模块

### 列表页（`/session`）

```
┌──────────────────────────────────────────────────┐
│  会话                                 [+ 新建]   │
├──────────────────────────────────────────────────┤
│  💬 my-session           KB: my-docs  2 条聊天   │
│  💬 test-session         KB: test-kb  0 条聊天   │
│                                                  │
└──────────────────────────────────────────────────┘
```

- 点击会话行 → 跳转 `/session/{name}`
- 每行右侧删除按钮

### 详情页（`/session/{name}`）— 核心页面

```
┌──────────────────────────────────────────────────────────────┐
│ ← 返回                                             会话名    │
├──────────────────────┬───────────────────────────────────────┤
│  会话: my-session    │  ┌─────────────────────────────────┐  │
│  知识库: my-docs     │  │  2026_06_25_14_30.json          │  │
│  [重新绑定]          │  │  ─────────────                   │  │
│                      │  │  👤 用户                         │  │
│  [＋ 新聊天]         │  │  什么是A1？                       │  │
│                      │  │  ─────────────                   │  │
│  📋 聊天列表         │  │  🤖 助手                         │  │
│  ▶ 2026_06_25_14_30  │  │  A1 是系统核心模块...             │  │
│  ▶ 2026_06_25_15_00  │  │                                 │  │
│  ▶ 2026_06_25_15_30  │  │                                 │  │
│                      │  │  ┌──────────────────── [↵]─┐   │  │
│                      │  │  │ 输入消息...             │提交│  │
│                      │  │  └─────────────────────────┘   │  │
│                      │  │                                 │  │
│                      │  └─────────────────────────────────┘  │
└──────────────────────┴───────────────────────────────────────┘
```

**左栏：**
- 当前会话名 + 知识库名（未绑定则显示 [绑定] 按钮）
- 新聊天按钮 → 调用 `POST /api/session/new`，右侧出现空白聊天框
- 聊天列表 → 点击切换，右侧展示该聊天内容

**右栏：**
- 消息展示区：最新消息在底部，鼠标滚轮上划加载历史
- 每条消息显示角色标识（用户/助手）和内容
- 输入框：Enter 提交，Shift+Enter 换行
- 提交按钮在输入框右侧

## 前端项目结构

```
ui/
├── index.html
├── package.json
├── vite.config.ts          # proxy /api → localhost:8000
├── tsconfig.json
└── src/
    ├── main.tsx
    ├── App.tsx             # 路由 + 导航栏
    ├── api/
    │   └── index.ts        # 封装所有 API 调用
    ├── pages/
    │   ├── KbList.tsx       # 知识库列表
    │   ├── KbDetail.tsx     # 知识库详情（文件管理）
    │   ├── SessionList.tsx  # 会话列表
    │   └── SessionChat.tsx  # 会话聊天（双栏）
    └── components/
        ├── NavBar.tsx       # 顶部导航
        ├── FileRow.tsx      # 文件/文件夹行
        └── ChatMessage.tsx  # 单条消息展示
```

## 技术选型

- React 18 + TypeScript
- Vite 构建
- React Router v6 路由
- CSS Modules 或 Tailwind CSS（可选）
- 无额外 UI 库依赖（保留轻量化）

## API 对接

前端通过 Vite proxy 调用后端：

```typescript
// ui/vite.config.ts
export default defineConfig({
  server: { proxy: { '/api': 'http://localhost:8000' } },
})
```

API 调用封装在 `src/api/index.ts` 中，每个后端端点对应一个函数。
